# 视觉元素库

按主题组织的图标和符号词汇。用作封面构图的构建模块。

## 科技与开发

| 元素 | 用途 |
|---------|---------|
| 代码窗口 / 终端 | 编程、开发 |
| 齿轮 / 齿轮组 | 工程、设置、流程 |
| 电路板 / 芯片 | 硬件、AI、计算 |
| 二进制 / 数据流 | 数据、算法 |
| API 括号 `</>` | Web 开发、API |
| 云 | 云计算、SaaS |
| 锁 / 盾牌 | 安全、隐私 |
| 网络节点 | 分布式系统、连接 |

## 渲染特定处理

| 渲染 | 元素风格 |
|-----------|---------------|
| `flat-vector` | 几何、简单形状、均匀填充 |
| `hand-drawn` | 草图感、有机、涂鸦风格 |
| `painterly` | 柔和边缘、笔触 |
| `digital` | 精确、渐变暗示、精致 |
| `pixel` | 8-bit 粗块、网格对齐 |
| `chalk` | 粉尘感、纹理、黑板风格 |
