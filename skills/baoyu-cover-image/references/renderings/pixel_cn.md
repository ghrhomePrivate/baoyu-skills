# pixel

复古 8-bit、怀旧、粗块

## 核心特征

像素艺术美学，带有可见的像素网格、有限的调色板和怀旧的游戏感。模拟经典的 8-bit 和 16-bit 时代图形。

## 线条

- 像素网格对齐，无抗锯齿
- 对角线上的阶梯边缘
- 单像素或双像素轮廓
- 方块状、棱角分明的形状

## 纹理

- 用于渐变的抖动图案
- 无平滑过渡
- 像素精度的交叉线
- 每个场景限制 16-32 色调色板

## 深度

- 无：仅平面像素平面
- 视差分层（前景/背景）
- 无透视或 3D 效果

## 元素词汇

- 8-bit 精灵和粗块形状
- 简单图标：星星、爱心、箭头
- 像素边框的文字气泡
- 粗块分段的进度条
- 复古游戏 UI 元素

## 字体方案

- 像素化位图字体风格
- 粗块方块字形
- 等宽或固定宽度感
- 标题使用全大写
