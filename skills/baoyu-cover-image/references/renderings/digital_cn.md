# digital

精致、精确、现代

## 核心特征

精致的数字插画，具有精细的打磨感、精确的边缘和微妙的现代效果。感觉像专业的 UI 模型或企业插画。

## 线条

- 干净、精确、计算机完美的边缘
- 一致的笔画粗细
- 适当位置使用锐利角
- 抗锯齿的平滑渲染

## 纹理

- 平滑表面，无可见纹理
- 允许微妙渐变（柔和、受控）
- 磨砂玻璃和模糊效果
- 方向一致的干净阴影

## 深度

- 微妙的渐变和柔和投影
- 分层的卡片式布局
- 轻微的 3D 效果（微妙，不写实）
- Material Design 风格的高度层次

## 元素词汇

- 精致的图标和 UI 组件
- 数据可视化：图表、图形、指标
- 卡片布局和结构化网格
- 标签芯片、进度条、状态指示器
- 干净的几何形状

## 字体方案

- 系统 UI 或现代无衬线字体（Inter、SF Pro 风格）
- 干净、功能性、高可读性
- 结构化的层次和一致的间距
