# flat-vector

简洁、现代、几何插画

## 核心特征

扁平设计，干净的轮廓、均匀的填充，无纹理或深度。如同现代应用图标、信息图插画和矢量编辑艺术。

## 线条

- 干净的轮廓，均匀的笔画粗细
- 封闭形状（涂色书风格）
- 圆润的线端，避免锐角
- 整体一致的笔画宽度

## 纹理

- 无：仅平滑、扁平的色彩填充
- 无渐变、阴影或噪点
- 纯色块

## 深度

- 扁平：无阴影、无透视
- 通过重叠实现的 2D 层次深度错觉
- 可选的 2.5D 等距分层（前后遮挡，无大气透视）
- 无 3D 效果或倒角

## 元素词汇

- 几何图标和简单形状
- 粗轮廓的物体配干净填充
- 几何简化：复杂物体 → 基本形状（树木 → 棒棒糖/三角形，建筑 → 矩形）
- "玩具模型"美学：可爱、圆润的比例
- 装饰性：圆点、线条、放射状线、药片形云朵、小星星
- 干净背景上的独立元素

## 字体方案

- 干净的无衬线或粗体几何字形
- 强可读性，一致的字重
- 任何尺寸下都可轻松缩放
