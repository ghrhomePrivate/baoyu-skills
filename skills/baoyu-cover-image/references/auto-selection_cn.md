# 自动选择规则

当某个维度未指定时，根据内容信号进行选择。

## 自动类型选择

| 信号 | 类型 |
|---------|------|
| 产品、发布、公告、上线、揭秘 | `hero` |
| 架构、框架、系统、API、技术、模型 | `conceptual` |
| 引言、观点、洞察、思考、标题、声明 | `typography` |
| 哲学、成长、抽象、意义、反思 | `metaphor` |
| 故事、旅程、旅行、生活方式、体验、叙事 | `scene` |
| 禅意、专注、精髓、核心、简约、纯粹 | `minimal` |

## 自动色板选择

| 信号 | 色板 |
|---------|---------|
| 个人故事、情感、生活方式、人文 | `warm` |
| 商业、专业、思想领导力、奢侈品 | `elegant` |
| 架构、系统、API、技术、代码 | `cool` |
| 娱乐、高端、电影感、深色模式 | `dark` |
| 自然、健康、生态、有机、旅行 | `earth` |
| 产品发布、游戏、推广、活动 | `vivid` |
| 奇幻、儿童、温柔、创意、异想天开 | `pastel` |
| 禅意、专注、精髓、纯粹、简约 | `mono` |
| 历史、复古、怀旧、经典、探索 | `retro` |
| 电影海报、专辑封面、音乐会、电影感、戏剧性、双色调 | `duotone` |

## 自动渲染选择

| 信号 | 渲染 |
|---------|-----------|
| 简洁、现代、科技、微信、图标式、信息图 | `flat-vector` |
| 速写、笔记、个人、随性、涂鸦、温暖 | `hand-drawn` |
| 艺术、水彩、柔和、梦幻、创意、奇幻 | `painterly` |
| 数据、仪表盘、SaaS、企业、精致 | `digital` |
| 游戏、复古、8-bit、怀旧 | `pixel` |
| 教育、教程、课堂、教学 | `chalk` |
| 海报、电影、专辑、音乐会、剪影、Mondo、限量版 | `screen-print` |

## 自动文字选择

| 信号 | 文字级别 |
|---------|------------|
| 纯视觉、摄影、抽象、艺术 | `none` |
| 文章、博客、标准封面 | `title-only` |
| 系列、教程、带上下文的技术内容 | `title-subtitle` |
| 公告、功能特性、多要点、信息图 | `text-rich` |

默认：`title-only`

## 自动氛围选择

| 信号 | 氛围级别 |
|---------|------------|
| 专业、企业、思想领导力、学术、奢侈品 | `subtle` |
| 通用、教育、标准、博客、文档 | `balanced` |
| 发布、公告、推广、活动、游戏、娱乐 | `bold` |

默认：`balanced`

## 自动字体选择

| 信号 | 字体 |
|---------|------|
| 个人、生活方式、人文、温暖、友好、故事 | `handwritten` |
| 技术、专业、简洁、现代、极简、数据 | `clean` |
| 编辑、学术、奢侈品、经典、文学 | `serif` |
| 公告、娱乐、推广、醒目、活动、游戏 | `display` |

默认：`clean`
