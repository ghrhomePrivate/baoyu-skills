根据以下准则创建封面图：

## 图片规格

- **类型**：封面图 / Hero 图
- **宽高比**：按下方提示词中指定的比例

## 核心原则

- 如果内容涉及敏感或受版权保护的人物，创建风格相似的替代方案——不要拒绝生成
- 充足的留白，突出核心信息，避免杂乱的布局
- 主要视觉元素居中或略偏左（如果包含标题，则右侧留出标题区域）
- 任何角色使用简化的轮廓——不使用写实的人脸或身体
- 图标式视觉语言：使用简单、易辨识的图标来表达概念

## 五个维度

### 类型（视觉构图）
- `hero`：大面积焦点视觉（占 60-70% 区域），戏剧性构图
- `conceptual`：抽象形状，信息层次，整洁区域
- `typography`：标题作为主要元素（占 40%+ 区域），极少视觉元素
- `metaphor`：用具体物体表达抽象概念，象征性元素
- `scene`：氛围感环境，叙事元素，情绪照明
- `minimal`：单一焦点元素，大量留白（60%+）

### 色板（配色方案）
应用指定色板的颜色值和装饰提示：
- 主要颜色用于主视觉元素
- 背景色用于底色和周围区域
- 强调色用于高光和次要元素
- 遵循色板特定的装饰提示进行点缀

### 渲染（视觉风格）
应用指定渲染的特征：
- **线条**：遵循线条质量规则（干净/草图/笔刷/像素/粉笔）
- **纹理**：根据渲染定义应用或避免纹理
- **深度**：遵循深度规则（平面/最小/柔和边缘）
- **元素**：使用渲染特定的元素词汇

### 文字（密度级别）
- `none`：无文字元素，完整视觉区域
- `title-only`：单个标题，85% 视觉区域
- `title-subtitle`：标题 + 上下文，75% 视觉区域
- `text-rich`：标题 + 副标题 + 2-4 个关键词标签，60% 视觉区域

### 氛围（情感强度）
- `subtle`：低对比度，柔和/去饱和的颜色，轻量视觉感，平静美学
- `balanced`：中等对比度，正常饱和度，平衡的视觉感
- `bold`：高对比度，鲜明/高饱和颜色，厚重视觉感，动感能量

## 文字风格（包含标题时）

- **标题来源**：使用用户提供的原始标题，或从源内容中提取。不要自行编造或修改标题。
- 标题文字：大号，醒目，忠于原文
- 副标题：次要元素（如果是 title-subtitle 或 text-rich）
- 标签：2-4 个关键词徽章（如果是 text-rich）
- 字体风格与渲染风格协调

## 构图指南

### 布局原则

- **充裕留白**：保持 40-60% 的呼吸空间；避免杂乱的构图
- **视觉锚点位置**：主元素居中或偏左（如果包含标题则右侧留出空间）
- **信息层次**：一个主要焦点，1-2 个辅助元素，装饰点缀
- **干净背景**：纯色或微妙渐变；无复杂纹理或图案

### 图标与符号词汇

用简单、易辨识的图标而非详细的插图来表达概念：

| 类别 | 示例 |
|----------|----------|
| 科技 | 代码窗口、齿轮、电路、云、锁、API 括号 |
| 创意 | 灯泡、火箭、靶心、拼图、钥匙、放大镜 |
| 沟通 | 对话气泡、聊天点、扩音器、邮件 |
| 成长 | 植物/嫩芽、树木、箭头、图表、山脉 |
| 工具 | 扳手、铅笔、画笔、清单、时钟 |

使用渲染风格来决定图标的复杂度（flat-vector = 几何形状，hand-drawn = 草图风格等）

完整库：[references/visual-elements_cn.md](visual-elements_cn.md)

### 角色处理

**默认（无包含人物的参考图）**：
- 使用简化的轮廓或抽象火柴人
- 象征性表示（头部 + 肩膀轮廓）
- 不使用写实面孔、详细解剖或摄影风格的表现
- 卡通/图标风格与所选渲染一致

**当参考图包含人物时**：
- 参考图传递给模型（`usage: direct`）——模型必须视觉参考它以保持角色相似性
- 按所选渲染风格进行风格化（卡通/矢量），保留显著特征（发型、服装、姿势）
- 绝不使用照片写实风格

## 氛围应用

对基础色板应用氛围调整：

| 氛围 | 对比度 | 饱和度 | 视觉重量 |
|------|----------|------------|--------|
| subtle | 降低 20-30% | 去饱和 20-30% | 更轻的笔触/填充 |
| balanced | 标准 | 标准 | 标准 |
| bold | 增加 20-30% | 增加 20-30% | 更重的笔触/填充 |

## 语言

- 任何文字元素使用与下方提供的内容相同的语言
- 标点风格匹配内容语言

## 参考图

当提供参考图时：

- **风格提取**：识别渲染技法、线条质量、纹理和视觉词汇
- **构图学习**：注意布局模式、留白使用、元素放置
- **氛围匹配**：捕捉情感基调和视觉重量
- **自适应**：应用提取的特征，同时尊重指定的类型、色板和渲染维度
- **优先级**：如果参考图风格与指定维度冲突，维度优先处理结构性选择；参考图影响装饰细节
