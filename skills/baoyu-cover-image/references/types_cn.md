# 类型构图指南

## 类型画廊

| 类型 | 描述 | 最适用于 |
|------|-------------|----------|
| `hero` | 大面积视觉冲击，标题覆盖 | 产品发布、品牌推广、重大公告 |
| `conceptual` | 概念可视化，抽象核心理念 | 技术文章、方法论、架构设计 |
| `typography` | 文字为主的布局，突出标题 | 观点文章、引言、洞察 |
| `metaphor` | 视觉隐喻，用具体表达抽象 | 哲学、成长、个人发展 |
| `scene` | 氛围场景，叙事感 | 故事、旅行、生活方式 |
| `minimal` | 极简构图，大面积留白 | 禅意、专注、核心概念 |

## 类型特定构图

| 类型 | 构图指南 |
|------|------------------------|
| `hero` | 大面积焦点视觉（占 60-70% 区域），标题覆盖在视觉上，戏剧性构图 |
| `conceptual` | 代表核心概念的抽象形状，信息层次，整洁区域 |
| `typography` | 标题作为主要元素（占 40%+ 区域），极少辅助视觉，强层次感 |
| `metaphor` | 用具体物体/场景表达抽象概念，象征性元素，情感共鸣 |
| `scene` | 氛围感环境，叙事元素，营造情绪的灯光和色彩 |
| `minimal` | 单一焦点元素，大面积留白（60%+），仅保留必要形状 |
