# 氛围维度

控制封面图的情感强度和视觉重量。

## 可选值

| 值 | 对比度 | 饱和度 | 重量 | 能量 |
|-------|:--------:|:----------:|:------:|:------:|
| `subtle` | 低 | 柔和 | 轻 | 平静 |
| `balanced` | 中 | 正常 | 中 | 适度 |
| `bold` | 高 | 鲜艳 | 重 | 动感 |

## 详细说明

### subtle

平静、低调的视觉呈现。

**特征**：
- 元素间低对比度
- 柔和、去饱和的颜色
- 轻量的视觉重量
- 温柔、精致的美感
- 柔和的边缘和过渡

**使用场景**：
- 思想领导力内容
- 专业/企业传播
- 冥想、健康主题
- 学术或学术文章
- 奢侈品牌美学

**配色指南**：
- 柔和色调、大地色、中性色
- 低饱和度（30-50%）
- 柔和渐变
- 最少的颜色种类（2-3 种颜色）

### balanced

通用、和谐的视觉呈现。

**特征**：
- 中等对比度
- 自然饱和度
- 平衡的视觉重量
- 清晰但不具攻击性
- 标准美学方案

**使用场景**：
- 一般文章（默认）
- 大多数博客内容
- 教育材料
- 产品文档
- 新闻和更新

**配色指南**：
- 标准饱和度（50-70%）
- 互补配色方案
- 清晰的前景/背景分离
- 适度的颜色种类（3-4 种颜色）

### bold

动感、高冲击力的视觉呈现。

**特征**：
- 元素间高对比度
- 鲜明、高饱和的颜色
- 厚重的视觉重量
- 充满活力、吸引注意
- 锐利的边缘和强烈的形状

**使用场景**：
- 产品发布
- 推广公告
- 活动营销
- 行动号召内容
- 娱乐/游戏主题

**配色指南**：
- 高饱和度（70-100%）
- 鲜艳的原色
- 强烈的对比度
- 动态的颜色组合（4+ 种颜色）

## 类型兼容性

| 类型 | subtle | balanced | bold |
|------|:------:|:--------:|:----:|
| hero | ✓ | ✓✓ | ✓✓ |
| conceptual | ✓✓ | ✓✓ | ✓ |
| typography | ✓ | ✓✓ | ✓✓ |
| metaphor | ✓✓ | ✓✓ | ✓ |
| scene | ✓✓ | ✓✓ | ✓ |
| minimal | ✓✓ | ✓✓ | ✗ |

✓✓ = 强烈推荐 | ✓ = 兼容 | ✗ = 不推荐

## 色板交互

氛围修改基础色板特征：

| 色板类别 | subtle | balanced | bold |
|------------------|--------|----------|------|
| 暖色板（warm, earth, pastel） | 更多留白、更柔和的色调 | 标准颜色 | 更深、更浓的暖色调 |
| 冷色板（cool, mono, elegant） | 更细的线条、柔和的颜色 | 标准颜色 | 更强的对比、更锐利的清晰度 |
| 深色板（dark, vivid） | 降低对比、柔和发光 | 标准颜色 | 最大冲击力、鲜艳饱和度 |
| 复古色板（retro） | 更多褪色、偏复古 | 标准颜色 | 更大胆的复古对比 |
| 双色调色板（duotone） | 双色间的柔和对比 | 标准双色分离 | 最大对比、鲜明分离 |

## 渲染交互

氛围调整渲染特征：

| 渲染 | subtle | balanced | bold |
|-----------|--------|----------|------|
| flat-vector | 更细的笔触、更浅的填充 | 标准重量 | 更粗的笔触、更强的填充 |
| hand-drawn | 更轻的笔压、更多留白 | 标准笔触 | 更重的马克笔笔触、更密集的元素 |
| painterly | 稀释的色彩、更多白色 | 标准笔触 | 更厚的颜料、饱和的笔触 |
| digital | 减少阴影、降低对比 | 标准渲染 | 更强的阴影、更锐利的边缘 |
| pixel | 更少颜色、更简单形状 | 标准色板 | 更多颜色、更密集的像素细节 |
| chalk | 更轻的粉笔、更多黑板可见 | 标准粉笔 | 厚重粉笔、鲜艳颜色、密集笔触 |
| screen-print | 更少颜色（2 种）、更浅的半调 | 标准 3-4 种颜色、中等半调 | 更多颜色（4-5 种）、密集半调、更强的套印偏移 |

## 自动选择

当 `--mood` 未指定时，根据信号选择：

| 信号 | 氛围级别 |
|---------|------------|
| 专业、企业、思想领导力、学术、奢侈品 | `subtle` |
| 通用、教育、标准、博客、文档 | `balanced` |
| 发布、公告、推广、活动、游戏、娱乐 | `bold` |

默认：`balanced`
