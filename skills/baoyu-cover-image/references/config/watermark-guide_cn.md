# 水印指南

## 位置示意图

```
┌─────────────────────────────┐
│                  [top-right]│
│                             │
│                             │
│       封面图                │
│                             │
│                             │
│[bottom-left][bottom-center][bottom-right]│
└─────────────────────────────┘
```

## 位置建议

| 位置 | 最适用于 | 避免使用的情况 |
|----------|----------|------------|
| `bottom-right` | 默认选择，最常用 | 标题在右下角 |
| `bottom-left` | 右侧偏重的布局 | 关键视觉元素在左下角 |
| `bottom-center` | 居中设计 | 底部文字较多的区域 |
| `top-right` | 底部偏重的内容 | 标题/头部在右上角 |

## 内容格式

| 格式 | 示例 | 风格 |
|--------|---------|-------|
| 账号 | `@username` | 社交媒体 |
| 域名 | `myblog.com` | 跨平台 |
| 品牌 | `MyBrand` | 简单品牌 |
| 中文 | `博客名` | 中文平台 |

## 最佳实践

1. **一致性**：所有封面使用相同的水印
2. **可读性**：确保水印在浅色和深色区域都可读
3. **尺寸**：保持低调——不应分散对内容的注意力

## 提示词集成

当水印启用时，添加到图片生成提示词中：

```
Include a subtle watermark "[content]" positioned at [position].
The watermark should be legible but not distracting from the main content.
```

## 封面特定注意事项

| 宽高比 | 推荐位置 | 备注 |
|--------------|---------------------|-------|
| 2.35:1 | bottom-right | 电影感——保持角落整洁 |
| 16:9 | bottom-right | 标准——位置灵活 |
| 1:1 | bottom-center | 正方形——居中通常效果更好 |

## 常见问题

| 问题 | 解决方案 |
|-------|----------|
| 水印不可见 | 调整位置或检查对比度 |
| 水印太突出 | 更改位置或缩小尺寸 |
| 水印与标题重叠 | 更改位置或缩小标题区域 |
