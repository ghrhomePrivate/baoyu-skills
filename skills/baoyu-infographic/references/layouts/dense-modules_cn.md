# dense-modules

高密度模块化布局，包含 6-7 个类型化的信息模块，填充了具体数据。

## 结构

- 每张图 6-7 个独立模块，每个模块承担特定的信息功能
- 每个模块包含具体数据：品牌名称、数字、百分比、参数
- 最小化留白——紧凑间距优先于呼吸空间
- 可接受较小文字以最大化信息密度
- 每个模块通过坐标标签或区域标记标识（如 MOD-1、SEC-A）

## 模块原型

| 模块 | 用途 | 内容要求 |
|--------|---------|---------------------|
| **品牌/选品阵列** | 带推荐的选项网格 | 4-8 个项目，配有图标、名称、简短描述；标出"最佳选择" |
| **参数刻度** | 质量/测量量表 | 3-5 个等级，带精确数值增量、质量指示器（表情、勾号） |
| **深度剖析/详解** | 关键项目的技术分解 | 放大标注、内部组件、剖面图或分解视图 |
| **场景对比** | 并排使用场景 | 3-6 个场景，每个场景有具体推荐和数据 |
| **辨别技巧** | 检查清单 | 3-5 种检验方法：看/测/查/问格式 |
| **警告/避坑区** | 必须避免的关键错误 | 3-5 个坑点及后果，1-2 个正确做法；高视觉对比度 |
| **速查手册** | 紧凑摘要 | 密集表格、一行摘要、决策流程图或关键要点 |

## 变体

| 变体 | 焦点 | 视觉重点 |
|---------|-------|-----------------|
| **坐标标注式** | 精确性和系统性 | 每个模块有字母数字坐标（A-01、B-05、C-12），标尺/轴标记 |
| **网格单元式** | 秩序和结构 | 模块在严格的矩形单元格中，由粗线分隔，瑞士网格风格 |
| **自由流动式** | 有机密度 | 杂志风格布局，虚线框，模块大小各异，箭头连接 |

## 最适用于

- 产品选购指南和购买指南
- 多维度对比内容
- 数据丰富的教育材料
- "避坑" / "完全指南"格式
- 面向小红书等高密度视觉需求平台的内容

## 视觉元素

- 模块边界标记（粗线、虚线框或坐标网格）
- 每个模块的质量指示器（表情、勾号、叉号、皇冠）
- 带突出数字的数据标注框
- 对比箭头和进度指示器
- 避坑模块的警告/提醒视觉标记
- 角落元数据（页码、时间戳、小条形码）

## 文字布局

- 主标题在顶部，醒目有冲击力
- 副标题带模块数量（"X大维度全面解析..."）
- 模块标题在彩色标签或带标签的框内
- 正文紧凑，模块内可用多栏
- 数字用强调色突出，略大于正文

## 信息密度规则

- 每个角落都应包含有用的信息或元数据
- 没有纯装饰性的空白
- 字号可以缩小以容纳更多内容——信息优先于字号
- 每个模块必须有具体的数据点，不能是笼统描述
- 在密度和可读性之间取得平衡：密集但有序

## 推荐搭配

- `pop-laboratory`：带坐标标记和蓝图网格的技术精度感
- `morandi-journal`：带涂鸦插图和有机框线的手绘温暖感
- `retro-pop-grid`：带严格网格单元和强烈对比的 1970 年代波普艺术
- `corporate-memphis`：适合产品对比的简洁商务感
- `technical-schematic`：适合技术产品指南的工程精度感
