# periodic-table

以统一单元格格式呈现的分类元素网格。

## 结构

- 矩形网格
- 每个单元格是一个元素
- 色彩编码分类
- 统一的单元格格式
- 可选的分组间隔

## 最适用于

- 分类集合
- 工具/资源目录
- 技能矩阵
- 元素合集
- 参考指南

## 视觉元素

- 统一的单元格大小
- 分类颜色
- 符号/缩写突出显示
- 每个单元格配有小图标
- 分类图例

## 文字布局

- 标题在顶部
- 单元格内：符号、名称、简要信息
- 图例中的分类名称
- 可选的行/列标题
- 特殊情况的脚注

## 推荐搭配

- `pop-art`：鲜艳的元素网格
- `pixel-art`：复古收藏展示
- `corporate-memphis`：商务工具目录
