# 结构化内容模板

用于生成结构化信息图内容以指导视觉设计的模板。

## 目的

本文档在内容分析与视觉设计之间起桥梁作用：
- 将源材料转换为设计师可用的格式
- 将学习目标组织为视觉章节
- 逐字保留所有源数据
- 将内容与设计指令分离

## 教学设计流程

### 第一阶段：高层大纲

1. **标题**：用一个引人注目的标题概括主旨
2. **概述**：简要描述（1-2 句话）
3. **学习目标**：列出观众将理解的内容

### 第二阶段：章节开发

针对每个学习目标：

1. **核心概念**：一句话概括该章节
2. **内容**：从源材料中逐字提取的要点
3. **视觉元素**：应该以视觉方式呈现的内容
4. **文字标签**：标题、副标题、标签的准确文字

### 第三阶段：数据完整性检查

验证所有源数据：
- 精确复制（无改述）
- 正确标注出处（引用类）
- 格式一致

## 关键规则

| 规则 | 要求 | 示例 |
|------|-------------|---------|
| **输出格式** | 仅 Markdown | 使用正确的标题、列表、代码块 |
| **语气** | 专家讲师 | 专业、清晰、有鼓励性 |
| **不添加新信息** | 仅使用源内容 | 不添加源材料中没有的示例 |
| **逐字数据** | 精确复制 | "73% 的增长"而非"显著增长" |

## 结构化内容格式

```markdown
# [信息图标题]

## 概述
[此信息图传达内容的简要描述 - 1-2 句话]

## 学习目标
观众将理解：
1. [主要目标]
2. [次要目标]
3. [第三目标（如适用）]

---

## 章节 1：[章节标题]

**核心概念**：[一句话概括本章节]

**内容**：
- [要点 1 - 逐字引用源材料]
- [要点 2 - 逐字引用源材料]
- [要点 3 - 逐字引用源材料]

**视觉元素**：[描述应以视觉方式呈现的内容]
- 类型：[图标/图表/插图/示意图/照片]
- 主题：[描绘什么]
- 处理方式：[应如何呈现]

**文字标签**：
- 标题："[标题的准确文字]"
- 副标题："[副标题的准确文字]"
- 标签："[标签 1]"、"[标签 2]"、"[标签 3]"

---

## 章节 2：[章节标题]

**核心概念**：[一句话概括]

**内容**：
- [要点 1]
- [要点 2]

**视觉元素**：[描述]

**文字标签**：
- 标题："[文字]"
- 标签："[标签 1]"、"[标签 2]"

---

[继续每个章节...]

---
```

## 按内容类型的章节模板

### 流程/步骤类

```markdown
## 章节 N：第 N 步 - [步骤标题]

**核心概念**：[此步骤要完成什么]

**内容**：
- 操作：[做什么]
- 详情：[怎么做]
- 备注：[重要注意事项]

**视觉元素**：
- 类型：编号步骤图标
- 主题：[代表该操作的视觉元素]
- 箭头：指向下一步

**文字标签**：
- 标题："第 N 步：[标题]"
- 操作："[动词 + 对象]"
```

### 对比类

```markdown
## 章节 N：[项目 A] vs [项目 B]

**核心概念**：[区分它们的关键点]

**内容**：
| 方面 | [项目 A] | [项目 B] |
|--------|----------|----------|
| [因素 1] | [值] | [值] |
| [因素 2] | [值] | [值] |

**视觉元素**：
- 类型：分屏对比
- 左侧：[项目 A 的表示]
- 右侧：[项目 B 的表示]

**文字标签**：
- 标题："[项目 A] vs [项目 B]"
- 左标签："[项目 A 名称]"
- 右标签："[项目 B 名称]"
```

### 层级类

```markdown
## 章节 N：[层级名称]

**核心概念**：[此层级代表什么]

**内容**：
- 位置：[顶层/中层/底层]
- 优先级：[重要性级别]
- 包含：[此层级的元素]

**视觉元素**：
- 类型：层/级
- 大小：[相对于其他层级]
- 位置：[在层级中的位置]

**文字标签**：
- 层级标题："[名称]"
- 描述："[简要描述]"
```

### 数据/统计类

```markdown
## 章节 N：[指标名称]

**核心概念**：[此数据展示什么]

**内容**：
- 数值：[精确数字/百分比]
- 上下文：[含义]
- 对比：[基准值（如有）]

**视觉元素**：
- 类型：[图表/数字突出/仪表盘]
- 强调方式：[如何吸引注意力]

**文字标签**：
- 主数字："[精确值]"
- 标签："[指标名称]"
- 上下文："[简要上下文]"
```

## 质量检查清单

在最终确定结构化内容之前：

- [ ] 标题是否传达了主要信息
- [ ] 学习目标是否清晰可衡量
- [ ] 每个章节是否对应一个目标
- [ ] 所有内容是否逐字引用源材料
- [ ] 视觉元素是否描述清晰
- [ ] 文字标签是否精确指定
- [ ] 数据点是否已收集并验证
- [ ] 设计指令是否已分离
- [ ] 是否未添加任何新信息
