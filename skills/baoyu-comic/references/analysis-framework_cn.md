# 漫画内容分析框架

将源内容转化为有效视觉叙事的深度分析框架。

## 目的

在创建漫画之前，彻底分析源材料以：
- 识别目标受众及其需求
- 确定漫画将提供的价值
- 提取视觉叙事的叙事潜力
- 规划角色弧线和关键时刻

## 分析维度

### 1. 核心内容（理解"是什么"）

**中心信息**
- 读者应该记住的最重要的一个观点是什么？
- 能用一句话表达吗？

**关键概念**
- 读者必须理解的基本概念有哪些？
- 这些概念应如何可视化？
- 哪些概念需要简化解释？

**内容结构**
- 源材料是如何组织的？
- 自然的叙事弧线是什么？
- 高潮和转折点在哪里？

**论据与示例**
- 哪些具体的例子、数据或故事支持主要观点？
- 哪些例子适合转化为视觉分格？
- 哪些内容可以用画面展示而非文字叙述？

### 2. 背景与语境（理解"为什么"）

**源头信息**
- 谁创作了这些内容？他们的视角是什么？
- 原始目的是什么？
- 是否存在需要注意的偏见？

**历史/文化背景**
- 故事发生在什么时间和地点？
- 读者需要哪些背景知识？
- 需要哪些时代特定的视觉元素？

**潜在假设**
- 源材料假设读者已经知道什么？
- 存在哪些隐含的信念或价值观？
- 漫画应该挑战还是强化这些假设？

### 3. 受众分析

**主要受众**
- 谁会阅读这部漫画？
- 他们现有的知识水平如何？
- 他们的兴趣和动机是什么？

**次要受众**
- 还有谁可能从这部漫画中受益？
- 他们的需求有何不同？

**读者疑问**
- 读者会有哪些疑问？
- 他们可能带有哪些误解？
- 我们可以创造哪些"恍然大悟"的时刻？

### 4. 价值主张

**知识价值**
- 读者将学到什么？
- 他们将获得哪些新视角？
- 这将如何改变他们的理解？

**情感价值**
- 读者应该感受到哪些情感？
- 他们将与角色建立怎样的连接？
- 什么会使这令人难忘？

**实用价值**
- 读者能否应用所学？
- 这可能激发哪些行动？
- 可能引发哪些讨论？

### 5. 叙事潜力

**故事弧线候选**
- 内容中存在哪些自然叙事？
- 冲突或张力在哪里？
- 发生了哪些转变？

**角色潜力**
- 关键人物是谁？
- 他们的动机和障碍是什么？
- 他们在整个过程中如何变化？

**视觉机会**
- 哪些场景具有强烈的视觉潜力？
- 哪里可以将抽象概念转化为具体画面？
- 哪些隐喻可以被可视化？

**戏剧性时刻**
- 突破/揭示时刻是什么？
- 情感高潮在哪里？
- 什么创造了张力和释放？

### 6. 改编考量

**保留内容**
- 关键事实和观点
- 重要引言或时刻
- 核心情感节拍

**简化内容**
- 复杂解释
- 密集技术细节
- 冗长描述

**扩展内容**
- 值得更多关注的简短提及
- 暗示的情感或关系
- 源材料中没有的视觉细节

**省略内容**
- 旁支信息
- 冗余示例
- 不服务于叙事的内容

## 输出格式

分析结果应保存到 `analysis.md`，包含：

1. **YAML Front Matter**：元数据（title、topic、time_span、source_language、user_language、aspect_ratio、recommended_page_count、recommended_art、recommended_tone、recommended_layout）
2. **目标受众**：主要、次要、第三受众及其需求
3. **价值主张**：读者将获得的价值（知识、情感、实用）
4. **核心主题**：包含主题、叙事潜力、视觉机会的表格
5. **关键人物与故事弧线**：角色档案，包含弧线、视觉形象、关键时刻
6. **内容信号**：基于内容类型的风格和布局推荐
7. **推荐方案**：按适合度排序的叙事方案

### YAML Front Matter 示例

```yaml
---
title: "Alan Turing: The Father of Computing"
topic: alan-turing-biography
time_span: 1912-1954
source_language: en
user_language: zh  # 来自 EXTEND.md 或检测结果
aspect_ratio: "3:4"
recommended_page_count: 16
recommended_art: ligne-claire  # ligne-claire|manga|realistic|ink-brush|chalk
recommended_tone: neutral      # neutral|warm|dramatic|romantic|energetic|vintage|action
recommended_layout: mixed      # standard|cinematic|dense|splash|mixed|webtoon
---
```

### 语言字段

| 字段 | 说明 |
|-------|-------------|
| `source_language` | 检测到的源内容语言 |
| `user_language` | 漫画输出语言（优先级：EXTEND.md > --lang > source_language） |

## 分析检查清单

进入分镜阶段之前：

- [ ] 能否用一句话说明核心信息？
- [ ] 是否确切知道谁会阅读这部漫画？
- [ ] 是否确定了至少 3 种漫画提供价值的方式？
- [ ] 是否有清晰的主角和引人入胜的弧线？
- [ ] 是否找到了至少 5 个视觉冲击力强的时刻？
- [ ] 是否了解了应保留、简化、扩展和省略的内容？
- [ ] 是否确定了情感的高峰和低谷？
