# warm

温馨基调 - 怀旧、亲切、温暖

## 概述

温暖、温馨的氛围，适合个人故事和怀旧内容。通过舒适的美学和治愈系视觉营造情感共鸣。

## 情绪特征

- 怀旧感
- 私密、温馨的氛围
- 治愈与抚慰
- 回忆与反思
- 柔和的情感温度

## 色彩修正

应用于任何画风时：

| 调整项 | 方向 |
|--------|------|
| 饱和度 | 略微降低 |
| 对比度 | 更柔和 |
| 色温 | 暖色偏移（+15%） |
| 亮度 | 柔和、金色调 |

## 色温

将调色板向暖色调偏移：

| 原始颜色 | 暖色偏移 |
|----------|----------|
| 冷蓝 | 柔和青绿 |
| 纯白 | 奶油色 |
| 灰色 | 暖灰 |
| 黑色 | 柔和炭灰 |

## 强调色

- 金黄色 (#D69E2E)
- 柔和橙色 (#DD6B20)
- 暖棕色 (#8B6F47)
- 日落色调

## 光照

- 黄金时刻光线
- 柔和、漫射光
- 温暖的室内光芒
- 烛光/灯光温暖
- 柔和阴影

## 情感表达范围

| 情感 | 表现 |
|------|------|
| 喜悦 | 真挚的温暖微笑 |
| 悲伤 | 淡淡的忧郁 |
| 爱 | 柔和、温柔的表情 |
| 回忆 | 远眺、沉思的目光 |

## 构图

- 亲密的取景
- 舒适的环境
- 柔焦背景
- 温馨的空间
- 突出私密时刻

## 视觉元素

- 温暖的光线
- 柔和的边缘
- 怀旧物品（旧照片、纪念品）
- 治愈物件（毛毯、茶杯）
- 自然元素（秋叶、夕阳）

## 适用场景

- 个人故事
- 童年回忆
- 师徒叙事
- 家族历史
- 温情传记
- 治愈之旅

## 组合说明

与以下画风搭配效果尤佳：
- ligne-claire：怀旧欧洲漫画
- realistic：感人至深的人物故事
- manga：日常温馨风
- chalk：怀旧教育风
