# action

动作基调 - 速度、冲击、力量

## 概述

高冲击力的动作氛围，具有动态运动、格斗效果和强大的视觉能量。创造震撼、激动人心的场景。

## 氛围特征

- 速度与动感
- 力量与冲击
- 格斗强度
- 体能能量
- 感官刺激

## 色彩调整

应用于任何画风时：

| 调整项 | 方向 |
|------------|-----------|
| 饱和度 | 高对比 |
| 对比度 | 最大 |
| 色温 | 因效果而异 |
| 亮度 | 动态范围 |

## 动作效果

**格斗/运动效果**（大量使用）：

| 效果 | 用途 |
|--------|-------|
| 速度线 | 运动、速度 |
| 冲击爆发 | 击打、碰撞 |
| 冲击波 | 强力打击 |
| 飞溅碎片 | 环境破坏 |
| 尘土云 | 地面冲击 |
| 动态模糊 | 快速移动 |
| 残影 | 超速 |

## 特殊效果

| 效果类型 | 视觉方式 |
|------------|-----------------|
| 能量攻击 | 发光、放射 |
| 物理冲击 | 放射线、碎片 |
| 运动 | 速度线、模糊 |
| 氛围 | 飞舞粒子、风 |

## 效果颜色

| 效果 | 颜色 | 色值 |

## 最适合

- 武术格斗
- 动作序列
- 体育时刻
- 体能挑战
- 战斗场景
- 高潮对决

## 组合说明

与以下搭配效果最佳：
- ink-brush：武侠格斗
- manga：少年战斗

避免搭配：
- chalk：风格不匹配
- ligne-claire：风格不匹配（过于静态）
