# neutral

中性基调 - 平衡、理性、教育性

## 概述

适用于教育和信息类内容的默认平衡基调。既不过度情感化也不冷漠——创造易于接受的专业氛围。

## 氛围特征

- 平衡的情感基调
- 清晰、理性的呈现
- 教育焦点
- 专业但平易近人
- 客观叙事

## 色彩调整

应用于任何画风时：

| 调整项 | 方向 |
|------------|-----------|
| 饱和度 | 标准（无偏移） |
| 对比度 | 平衡 |
| 色温 | 中性 |
| 亮度 | 略微明亮 |

## 光照

- 均匀、清晰的光照
- 最少的戏剧性阴影
- 各分格间一致
- 自然光源
- 无极端对比

## 情感范围

| 情感 | 表达程度 |
|---------|-----------------|
| 喜悦 | 适度微笑 |
| 关切 | 若有所思的表情 |
| 惊讶 | 眼睛略微睁大 |
| 沮丧 | 微微皱眉 |

## 构图

- 平衡的分格布局
- 清晰的焦点
- 可读的层次
- 标准取景
- 功能性构图

## 最适合

- 教育内容
- 技术教程
- 信息类传记
- 纪录片风格
- 专业主题

## 使用说明

Neutral 是默认基调。与任何画风组合可获得基线专业输出。最具通用性的基调选项。
