# vintage

复古基调 - 历史感、陈旧、年代真实感

## 概述

带有陈旧纸张效果和年代对应美学的历史氛围。创造时间感、真实性和历史距离感。

## 氛围特征

- 历史真实感
- 年代距离感
- 档案品质
- 时间与记忆
- 古典优雅

## 色彩调整

应用于任何画风时：

| 调整项 | 方向 |
|------------|-----------|
| 饱和度 | 降低、柔和 |
| 对比度 | 中等、老化 |
| 色温 | 棕褐偏移 |
| 亮度 | 略微褪色 |

## 色板

偏向陈旧色调：

| 角色 | 颜色 | 色值 |
|------|-------|-----|
| 主色 | 棕褐 | #8B7355 |
| 背景 | 老化纸张 | #F5E6D3 |
| 强调 1 | 褪色青 | #6B8E8E |
| 强调 2 | 柔和酒红 | #7B3F3F |
| 墨色 | 老化黑 | #3D3D3D |
| 泛黄 | 纸张黄 | #F5DEB3 |

## 视觉效果

**老化效果**（微妙应用）：

| 效果 | 应用 |
|--------|-------------|
| 纸张老化 | 背景纹理 |
| 褪色边缘 | 暗角效果 |
| 灰尘斑点 | 微妙叠加 |
| 泛黄 | 色彩偏移 |
| 磨损痕迹 | 边角细节 |

## 最适合

- 1950 年代之前的故事
- 古典科学史

- 档案叙事

## 组合说明

与以下搭配效果最佳：
- realistic：年代剧
- ligne-claire：历史冒险
- ink-brush：古典亚洲故事

避免搭配：
- manga：风格不匹配（过于现代）
- chalk：风格不匹配（现代教育风）
