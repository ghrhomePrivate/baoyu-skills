# romantic

浪漫基调 - 柔和、唯美、情感细腻

## 概述

柔和、梦幻的氛围，适合浪漫和情感细腻的内容。以装饰元素、闪光和美丽构图为特征，强调感觉和美感。

## 氛围特征

- 浪漫与爱情
- 美丽与优雅
- 情感细腻
- 梦想与希望
- 青春与理想主义

## 色彩调整

应用于任何画风时：

| 调整项 | 方向 |
|------------|-----------|
| 饱和度 | 柔和粉彩 |
| 对比度 | 低、柔和 |
| 色温 | 略偏粉色暖调 |
| 亮度 | 柔和、发光 |

## 色板

偏向浪漫色调：

| 角色 | 颜色 | 色值 |
|------|-------|-----|
| 主色 | 柔粉 | #FFB6C1 |
| 副色 | 薰衣草 | #E6E6FA |
| 强调 | 玫瑰 | #FF69B4 |

| 金色 | 金色闪光 | #FFD700 |
| 肤色 | 瓷白 | #FFF5EE |

## 光照

- 柔和、漫射光
- 发光效果
- 逆光光晕
- 闪光高光
- 梦幻氛围

## 装饰元素

**必要装饰**（添加到构图中）：

| 元素 | 用途 |
|---------|-------|
| 花瓣 | 漂浮、构图框架 |
| 闪光 | 情感高光 |
| 气泡 | 梦幻时刻 |
| 羽毛 | 轻柔漂浮 |
| 星星 | 夜景、惊奇 |
| 爱心 | 爱情强调 |
| 光晕 | 角色高光 |

## 情感范围

| 情感 | 表达 |
|---------|-----------|
| 爱情 | 柔和目光，脸红 |
| 思念 | 远方的、美丽的忧伤 |
| 喜悦 | 灿烂笑容，闪光 |
| 害羞 | 低垂的眼睛，脸红 |

## 构图

- 优雅、流畅的布局
- 柔焦背景
- 装饰框架中的角色
- 美丽角度（3/4 侧脸）
- 网点渐变

## 最适合

- 恋爱故事
- 成长故事
- 友情叙事
- 情感剧
- 校园生活
- 美丽时刻

## 组合说明

与以下搭配效果最佳：
- manga：经典少女风格

避免搭配：
- realistic：风格不匹配
- ink-brush：风格不匹配
- ligne-claire：风格不匹配
- chalk：风格不匹配
