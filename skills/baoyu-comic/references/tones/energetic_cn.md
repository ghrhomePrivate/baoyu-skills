# energetic

活力基调 - 明亮、动感、激动人心

## 概述

充满活力的高能量氛围，适合令人兴奋、充满发现的内容。明亮的色彩、动感的构图和运动感为年轻受众创造引人入胜的视觉效果。

## 氛围特征

- 兴奋与惊奇
- 发现与学习
- 能量与热情
- 运动与动作
- 青春精神

## 色彩调整

应用于任何画风时：

| 调整项 | 方向 |
|------------|-----------|
| 饱和度 | 高（鲜艳） |
| 对比度 | 中高 |
| 色温 | 多变、有冲击力 |
| 亮度 | 明亮、干净 |

## 色板

偏向鲜艳色调：

| 角色 | 颜色 | 色值 |
|------|-------|-----|
| 主红 | 亮红 | #F56565 |
| 主黄 | 阳光黄 | #F6E05E |
| 主蓝 | 天蓝 | #63B3ED |
| 强调 1 | 品红 | #D53F8C |
| 强调 2 | 青柠绿 | #68D391 |
| 背景 | 干净白 | #FFFFFF |
| 备选背景 | 明亮粉彩 | 多种 |

## 光照

- 明亮、清晰的光照
- 干净的阴影
- 高能量
- 聚光灯效果用于强调
- 动态光源

## 动态元素

**能量效果**（添加到构图中）：

| 元素 | 用途 |
|---------|-------|
| 速度线 | 运动、兴奋 |
| 闪光 | 发现 |
| 爆发效果 | 恍然大悟时刻 |
| 动态模糊 | 快速动作 |
| 星爆 | 强调 |
| 汗滴 | 努力/惊讶 |

## 情感范围

| 情感 | 表达 |
|---------|-----------|
| 兴奋 | 瞪大眼睛，灿烂笑容 |
| 惊讶 | 夸张反应 |
| 决心 | 强烈专注 |
| 惊奇 | 闪闪发光的眼睛 |

## 构图

- 动态角度
- 动作导向布局
- 运动强调
- 干净、有力的设计
- 能量流动

## 视觉风格

- 富有表现力的动画角色
- 大眼睛，大反应
- 动态姿势
- 运动和动作焦点
- 简化背景保持能量

## 最适合

- 科学解说
- "恍然大悟"时刻
- 年轻受众内容
- 发现叙事
- 学习冒险
- 动作教程

## 组合说明

与以下搭配效果最佳：
- manga：少年能量
- chalk：趣味教育

避免搭配：
- realistic：风格不匹配
- ink-brush：风格不匹配
