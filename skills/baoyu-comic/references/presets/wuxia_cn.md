# wuxia

武侠预设 - 港式武侠漫画风格

## 基础配置

| 维度 | 值 |
|-----------|-------|
| 画风 | ink-brush |
| 基调 | action |
| 布局 | splash（默认） |

等价于：`--art ink-brush --tone action`

## 独特规则

此预设包含超出画风+基调组合的特殊规则。当使用 `--style wuxia` 时，必须应用以下所有规则。

### 气/能量效果（必需）

武术力量必须通过气效果可见：

| 效果类型 | 视觉处理 |
|-------------|-----------------|
| 内气 | 角色周围的发光气场 |
| 外气 | 可见的能量投射 |
| 气碰撞 | 放射冲击波 |
| 吸气 | 粒子向角色流动 |
| 隐藏力量 | 眼睛/拳头微妙发光 |

### 能量颜色

| 气类型 | 颜色 |
|---------|-------|
| 正气 | 蓝色（#4299E1）、金色（#FFD700） |
| 刚猛 | 红色（#DC2626）、橙色（#EA580C） |
| 邪气 | 紫色（#7C3AED）、绿色（#16A34A） |
| 纯净 | 白色、银色 |
| 远古 | 金色配粒子 |

### 格斗视觉语言

**冲击时刻**必须包含：

1. 从冲击点放射的速度线
2. 飞溅的碎片（石头、木材、布料）
3. 冲击波环
4. 尘埃/能量云
5. 头发和衣物被吹起

### 动作描绘

| 速度等级 | 视觉处理 |
|-------------|-----------------|
| 正常 | 标准姿势 |
| 快速 | 动态模糊、速度线 |
| 闪电 | 残影、多重位置 |
| 瞬移 | 渐隐效果、粒子轨迹 |

### 环境融合

背景必须支持动作：

| 环境 | 格斗增强 |
|-------------|-------------------|
| 山脉 | 冲击导致山峰崩塌 |
| 森林 | 树木爆裂、落叶飞舞 |
| 水面 | 壮观的水花、水上行走 |
| 寺庙 | 柱子断裂、瓦片飞舞 |
| 悬崖 | 戏剧性坠落、风效果 |

### 角色姿势指南

- 带重心分配的动态武者姿势
- 展示运动的飘逸袍衣和头发
- 动作中可见的肌肉张力
- 双脚站定或动态运动中
- 传统武术姿势

### 武器效果

| 武器 | 视觉处理 |
|--------|-----------------|
| 剑 | 拖尾光弧、剑身发光 |
| 掌 | 气投射、风效果 |
| 棍 | 旋转模糊、冲击涟漪 |
| 鞭 | 流动能量轨迹 |

### 氛围元素

始终包含：
- 漂浮粒子（树叶、花瓣、尘埃）
- 水墨渲染的雾气增添纵深
- 风向指示
- 适当时的戏剧性天空/天气

## 质量标记

- ✓ 具有动感的动态动作姿势
- ✓ 线条中的水墨笔触美学
- ✓ 可见的气/能量效果
- ✓ 高对比度戏剧性光照
- ✓ 带中国元素的大气背景
- ✓ 飘逸的衣物和头发动态
- ✓ 冲击力强的格斗时刻
- ✓ 速度线和冲击效果
