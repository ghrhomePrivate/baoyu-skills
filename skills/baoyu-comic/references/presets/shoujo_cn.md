# shoujo

少女预设 - 经典少女漫画，浪漫美学

## 基础配置

| 维度 | 值 |
|-----------|-------|
| 画风 | manga |
| 基调 | romantic |
| 布局 | standard（默认） |

等价于：`--art manga --tone romantic`

## 独特规则

此预设包含超出画风+基调组合的特殊规则。当使用 `--style shoujo` 时，必须应用以下所有规则。

### 装饰元素（必需）

每个情感时刻必须包含装饰元素：

| 情感 | 必需装饰 |
|---------|---------------------|
| 爱情 | 漂浮的爱心、闪光、玫瑰花瓣 |
| 思念 | 羽毛、气泡、远处的闪光 |
| 喜悦 | 花朵绽放、光芒迸射、星星 |
| 悲伤 | 飘落的花瓣、消逝的闪光 |
| 害羞 | 柔和闪光、漂浮气泡 |
| 顿悟 | 放射线配闪光 |

### 眼部细节要求

眼睛在少女风格中至关重要：

| 方面 | 处理方式 |
|--------|-----------|
| 大小 | 比标准漫画更大（1.2 倍） |
| 高光 | 多个（3-5 个），根据情感放置 |
| 反射 | 情感时刻眼中映射场景 |
| 闪光 | 内置闪光效果 |
| 泪水 | 水晶般、精细的泪滴 |

### 角色美感标准

| 特征 | 处理方式 |
|---------|-----------|
| 头发 | 飘逸、精细的发丝、光泽高光 |
| 皮肤 | 瓷白，脸颊柔和腮红 |
| 嘴唇 | 柔和、略带光泽 |
| 手部 | 优雅、富有表现力的手势 |
| 姿态 | 优雅、端庄的姿势 |

### 背景效果

情感时刻使用**抽象背景**：

| 时刻类型 | 背景 |
|-------------|-----------|
| 告白 | 柔和渐变 + 漂浮花朵 |
| 震惊 | 网点速度线 + 闪光 |
| 回忆 | 梦幻模糊 + 散落花瓣 |
| 顿悟 | 放射线 + 光爆 |
| 亲密 | 柔焦 + 漂浮元素 |

### 分格流向

- 亲密时刻分格重叠
- 为情感冲击打破分格边框
- 装饰元素在分格之间漂浮
- 使用网点渐变营造氛围
- 不规则分格形状增添戏剧性

### 情感节拍节奏

为情感冲击放慢节奏：

| 场景类型 | 分格处理 |
|------------|-----------------|
| 告白 | 多个小分格，然后大图 |
| 对视 | 特写序列 |
| 触碰 | 慢动作分格分解 |
| 顿悟 | 铺垫分格后冲击 |

### 色板应用

| 场景类型 | 色板 |
|------------|---------|
| 浪漫 | 粉色、薰衣草、玫瑰金 |
| 欢乐 | 柔和黄、桃色、天蓝 |
| 悲伤 | 淡蓝、银色、灰薰衣草 |
| 戏剧性 | 深玫瑰、紫色、对比色 |

### 网点使用

| 氛围 | 网点图案 |
|------|-------------|
| 中性 | 干净、极简 |
| 浪漫 | 柔和渐变叠加 |
| 戏剧性 | 重对比网点 |
| 梦幻 | 柔和圆点图案 |
