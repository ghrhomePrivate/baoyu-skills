# manga

日漫画风 - 动漫/漫画美学，表现力丰富的角色

## 概述

日本漫画画风，以大而富有表现力的眼睛、动态姿势和视觉情感符号为特征。用途广泛的风格，适用于从教育到浪漫到动作的各种类型。

## 线条处理

- 干净、流畅的线条（1.5-2px）
- 富有表现力的粗细变化用于强调
- 流畅曲线，动感笔触
- 可用速度线和动态效果
- 网点效果营造氛围

## 角色设计

- 动漫/漫画比例：较大的眼睛，丰富的表情
- 5-7 个头身比例（因子风格而异）
- 清晰的情感符号（！、？、汗滴、闪光）
- 动态姿势和手势
- 精细的发丝描绘
- 时尚的服装，自然的褶皱

## 眼睛风格

| 类型 | 说明 |
|------|-------------|
| 标准 | 中等偏大，2-3 个高光点 |
| 教育 | 友好、平易近人的眼睛 |
| 戏剧 | 强烈、精细的虹膜 |
| 可爱 | 非常大、闪亮的眼睛 |

## 背景处理

- 对话/解说时简化处理
- 建立镜头时详细描绘
- 网点渐变营造氛围
- 情感时刻使用抽象背景
- 技术图表以显示屏风格呈现

## 色彩方式

- 干净、明亮的动漫色彩
- 皮肤上柔和的渐变
- 鲜艳的色板选择
- 光影柔和过渡
- 色彩编码用于角色识别
