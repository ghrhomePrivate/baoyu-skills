## 封面设计原则

- 学术庄重与视觉吸引力兼顾
- 标题排版体现知识/科学主题
- 构图暗示核心主题（人物剪影、标志性符号、概念图）
- 副标题或时间跨度展现史诗格局

## 分格构图指南

| 分格类型 | 推荐数量 | 用途 |
|-----------|-------------------|-------|
| 主叙事 | 每页 3-5 | 故事推进 |
| 概念图解 | 每页 1-2 | 可视化抽象概念 |
| 叙述者分格 | 每页 0-1 | 评述、过渡 |
| 大图（整页/半页） | 偶尔 | 重要时刻 |

## 分格尺寸参考

- **整页（大图）**：重要时刻、关键突破
- **半页**：重要场景、转折点
- **1/3 页**：标准叙事分格
- **1/4 或更小**：快速推进、连续动作

## 概念可视化技巧

将抽象概念转化为具体视觉：

| 抽象概念 | 视觉方式 |
|-----------------|-----------------|
| 神经网络 | 发光节点与连接线 |
| 梯度下降 | 球在山谷地形上滚动 |
| 数据流 | 在管道中流动的发光粒子 |
| 算法迭代 | 上升的螺旋阶梯 |
| 突破时刻 | 打破屏障、穿透之光 |
| 逻辑证明 | 拼搭的积木 |
| 不确定性 | 分岔路径、迷雾、多重影子 |

## 文字元素设计

| 文字类型 | 风格 | 用途 |
|-----------|-------|-------|
| 角色对白 | 椭圆对话气泡 | 主叙事对话 |
| 旁白评述 | 矩形框 | 解释、评述 |
| 说明条 | 边缘矩形条 | 时间、地点信息 |
| 思考气泡 | 云朵形状 | 角色内心独白 |
| 术语标注 | 粗体 / 特殊颜色 | 技术术语首次出现 |
