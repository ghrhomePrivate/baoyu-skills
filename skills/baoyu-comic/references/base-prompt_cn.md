按照以下指南创建知识传记漫画页面：

## 图像规格

- **类型**：多分格漫画书页面
- **方向**：竖版（垂直）
- **宽高比**：2:3
- **风格**：参见特定风格参考文件获取视觉指南

## 分格结构

### 分格边框
- 每个分格周围干净的黑色线条（1-2px）
- 分格之间白色间距（8-12px）
- 分格排列确保清晰的阅读流
- 分格大小变化创造视觉节奏

### 分格构图
- 每个分格有清晰的焦点
- 正确使用前景、中景、背景
- 机位角度变化：平视、俯视、仰视、特写、全景
- 动作在分格之间逻辑流畅
- 有意使用留白

## 文字元素

### 对话气泡
- **对白**：椭圆形气泡带尖角指示
- 白色填充配细黑色轮廓
- 尖角清晰指向说话者
- 手写体风格字体（非电脑生成）

### 旁白框
- **旁白/叙述者**：矩形框
- 通常位于分格边缘（顶部或底部）
- 略微不同的填充色（米色或浅黄色）
- 用于评述、时间跳跃、解释

### 思考气泡
- 云朵形状，气泡轨迹指向思考者
- 比对话气泡更柔和的轮廓
- 用于内心独白

### 说明条
- 分格边缘的矩形条
- 时间和地点信息
- "与此同时……"、"三年后……"类过渡
- 深色填充配白色文字，或反之

### 文字排版
- 全程手绘字体风格
- 粗体用于强调和关键术语
- 一致的字号
- 中文文字：使用全角标点""，。！
- 清晰的层次：标题 > 对白 > 说明

## 科学/概念可视化

描绘抽象概念时：

| 概念 | 视觉隐喻 |
|---------|----------------|
| 神经网络 | 发光节点由干净线条连接 |
| 数据流 | 沿简洁路径流动的发光粒子 |
| 算法 | 几何图案，积木拼搭 |
| 逻辑/证明 | 咬合的拼图块 |
| 发现 | 光芒穿透黑暗 |
| 不确定性 | 分岔路径，问号 |
| 时间 | 时钟元素，日历页 |

- 将图表自然融入叙事分格
- 使用嵌入分格或思考气泡风格进行解释
- 简化的图标优于写实描绘

## 打破第四面墙 / 叙述者角色

描绘叙述者角色向读者说话时：
- 角色可以直视画面外
- 可以出现在"当下"框架场景中
- 与主时间线有明显不同的视觉处理
- 通常在页面边缘或专用分格中
- 可以评论或质疑所展示的事件

## 历史准确性

- 研究时代特定细节：服装、科技、建筑
- 跨时间段自然展示角色的衰老
- 标志性物品和地点以可辨识方式呈现
- 在准确性和风格化之间取得平衡
