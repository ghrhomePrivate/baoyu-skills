# Screen-Print 风格

大胆的 poster 艺术：半色调纹理、有限色数与符号化叙事。

## 元素组合

```yaml
canvas:
  ratio: portrait-3-4
  grid: single | dual

image_effects:
  cutout: silhouette
  stroke: none
  filter: halftone | print-grain

typography:
  decorated: stroke-text | shadow-3d
  tags: none
  direction: horizontal

decorations:
  emphasis: star-burst | numbering
  background: solid-saturated | paper-texture
  doodles: none
  frames: none
```

## 色板

| 角色 | 颜色 | Hex |
|------|--------|-----|
| 主色 | Burnt Orange、深青绿 | #E8751A, #0A6E6E |
| 背景 | 近黑、暖奶油 | #121212, #F5E6D0 |
| 点缀 | Crimson、琥珀 | #C0392B, #F4A623 |

**双色搭配**（按内容情绪任选其一）：

| 组合 | 色 A | 色 B | 气质 |
|------|---------|---------|------|
| Orange + Teal | #E8751A | #0A6E6E | 电影感、动感 |
| Red + Cream | #C0392B | #F5E6D0 | 大胆、经典 |
| Blue + Gold | #1A3A5C | #D4A843 | 高级、尊贵感 |
| Crimson + Navy | #DC143C | #0D1B2A | 戏剧感、黑色电影风 |
| Magenta + Cyan | #C2185B | #00BCD4 | 鲜艳、 pop |

**规则**：最多使用 2–5 种颜色。颜色越少，冲击力越强。

## 视觉元素

- 大胆的剪影与符号化形状
- 色块填充内的半色调网点
- 轻微色层错位（印刷偏移感）
- 几何构图框（圆、拱、三角）
- 图底反转（负形讲述次要故事）
- stencil 切割边缘、无描边——靠色块边界塑形
- 字体作为设计元素融入画面，而非简单叠字
- 复古海报式边框处理

## 字体与排版

- 粗窄体 sans-serif 或手绘字
- Art Deco 影响、复古海报字体设计
- 字体是构图的有机部分（非单独一层）
- 与背景高对比以保证可读性

## 最佳版式搭配

| 版式 | 适配度 | 适用场景 |
|--------|---------------|----------|
| sparse | ✓✓ | 标志性 poster 封面、强态度陈述 |
| balanced | ✓✓ | 编辑式构图、观点文 |
| dense | ✗ | 信息过多会与极简 poster 美学冲突 |
| list | ✓ | 大胆排行、精选 |
| comparison | ✓✓ | 双色分屏、before/after |
| flow | ✓ | 电影感推进、时间线 |
| mindmap | ✗ | 对几何 poster 风格过于复杂 |
| quadrant | ✓✓ | 强几何分区、分类 |

## 最适合

- 观点文、文化评论
- 电影/音乐/书籍推荐
- 戏剧性宣告
- Before/after 转变
- 大胆的 editorial 内容
- 活动推广
