# Study Notes 风格

写实手写笔记照片感——学生笔记风，密集略乱但可读。

## 元素组合

```yaml
canvas:
  ratio: portrait-3-4
  grid: single

image_effects:
  cutout: none
  stroke: none
  filter: natural-photo

typography:
  decorated: none
  tags: none
  direction: horizontal

decorations:
  emphasis: circle-mark | underline | checkmark | cross | star-simple
  background: lined-paper-white
  doodles: arrows-simple | margin-notes | corrections | explanatory-diagrams
  frames: none
```

## 色板（三色批注体系）

| 角色 | 颜色 | Hex |
|------|--------|-----|
| 主色 | 蓝色圆珠笔、黑墨水 | #1E3A5F, #1A1A1A |
| 高亮 | 黄色荧光笔 | #FFFF00（50% 透明度） |
| 点缀 | 红笔（圈画、下划线） | #CC0000 |
| 背景 | 白横格纸 | #FFFFFF |

## 视觉元素

- 写实照片视角：俯视书桌
- 手持蓝色圆珠笔、正在划线
- 手写内容极密，铺满整页
- 红笔批注：圈、下划线、星、方框
- 黄色荧光笔标关键词
- 涂改痕迹、挤在页边的批注
- 简单手绘符号：→ * ✓ ✗ !
- 笔压变化形成深浅笔触

## 字体与排版

- 真实学生手写感
- 略乱但可读、结构清晰
- 字号多变（大标题、正文、极小边注）
- CJK 优化

## 内容结构

三段式布局：

### 上区
- 核心主题（红笔多次圈出）
- 第一节标题 + 3–4 个要点
- 箭头连接、红笔下划线

### 中区
- 第二节标题（红笔方框）
- 编号步骤 ①②③
- 具体方法与补充说明

### 下区
- 第三节标题（红星）
- 时间点 / 关键指标
- 金句 / 核心提示（角落极小字）

## 最佳版式搭配

| 版式 | 适配度 | 适用场景 |
|--------|---------------|----------|
| sparse | ✗ | 不适用——本风格需要密集内容 |
| balanced | ✓ | 内容较轻时 |
| dense | ✓✓ | 最契合——知识笔记、摘要 |
| list | ✓✓ | 步骤清单、排行 |
| comparison | ✓ | 对比分析 |
| flow | ✓ | 流程 |
| mindmap | ✓✓ | 思维导图笔记 |
| quadrant | ✓ | 四象限分析 |

## 最适合

- 复习资料、考试笔记
- 知识整理、框架总结
- 教程摘要、速记
- 「学霸笔记」风内容
- 需要真实感的知识分享

## 风格规则

### 建议 ✓
- 保持内容极度密集
- 使用简单符号（→ * ✓ ✗ !）
- 用红笔标注重点
- 包含涂改痕迹
- 把极小字挤进页边

### 避免 ✗
- 使用复杂 emoji
- 留白过多
- 过于整齐、洁癖式排版
- 花哨彩色装饰
- 卡通元素
