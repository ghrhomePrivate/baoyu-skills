# Warm 风格

温馨、友好、易亲近的视觉风格。

## 元素组合

```yaml
canvas:
  ratio: portrait-3-4
  grid: single | dual

image_effects:
  cutout: soft
  stroke: white-solid | glow
  filter: warm-tone | cream-skin

typography:
  decorated: highlight | handwritten
  tags: ribbon | bubble
  direction: horizontal

decorations:
  emphasis: star-burst | hearts
  background: solid-pastel | gradient-radial
  doodles: clouds | stars-sparkles
  frames: polaroid | tape-corners
```

## 色板

| 角色 | 颜色 | Hex |
|------|--------|-----|
| 主色 | 暖橙、金黄、陶土色 | #ED8936, #F6AD55, #C05621 |
| 背景 | 奶油色、柔和桃 | #FFFAF0, #FED7AA |
| 点缀 | 深棕、柔和红 | #744210, #E57373 |

## 视觉元素

- 阳光射线、咖啡杯、治愈小物
- 暖光效果
- 友好、有邀请感的装饰
- 柔和、舒适的形状

## 字体与排版

- 友好、偏圆的手写体
- 暖色点缀
- 舒适、好接近的气质

## 最佳版式搭配

| 版式 | 适配度 | 适用场景 |
|--------|---------------|----------|
| sparse | ✓✓ | 情绪向封面 |
| balanced | ✓✓ | 个人故事 |
| dense | ✓ | 细腻经历 |
| list | ✓ | 人生经验清单 |
| comparison | ✓✓ | Before/after 故事 |
| flow | ✓ | 历程叙事 |

## 最适合

- 个人故事
- 人生经验、教训
- 情绪向内容
- 舒适与生活方式
- 真诚分享
