# Minimal 风格

极净、克制、高级感。

## 元素组合

```yaml
canvas:
  ratio: portrait-3-4
  grid: single

image_effects:
  cutout: clean
  stroke: none | white-solid
  filter: none | muted-tones

typography:
  decorated: none
  tags: white-black | pill
  direction: horizontal

decorations:
  emphasis: underline | circle-mark
  background: solid-white | solid-pastel
  doodles: hand-drawn-lines
  frames: none | rounded-rect
```

## 色板

| 角色 | 颜色 | Hex |
|------|--------|-----|
| Primary | 黑、白 | #000000, #FFFFFF |
| Background | 米白、纯白 | #FAFAFA, #FFFFFF |
| Accents | 单色（随内容提取） | 蓝、绿或珊瑚色 |

## 视觉元素

- 单一焦点、细线条
- 留白最大化
- 装饰极简、克制
- 元素少而精

## 字体

- 干净、简约手写
- 字重变化少
- 字距优雅

## 布局搭配建议

| 布局 | 适配度 | 用途 |
|--------|---------------|----------|
| sparse | ✓✓ | 优雅金句 |
| balanced | ✓✓ | 专业向内容 |
| dense | ✓✓ | 干净知识卡 |
| list | ✓ | 简单列表 |
| comparison | ✓ | 清爽对比 |
| flow | ✓ | 克制流程图 |

## 适合场景

- 专业内容
- 严肃话题
- 优雅展示
- 高端产品
- 商务向内容
