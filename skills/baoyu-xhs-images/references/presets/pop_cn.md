# Pop 风格

鲜艳、有活力、抓眼球的视觉风格。

## 元素组合

```yaml
canvas:
  ratio: portrait-3-4
  grid: single | quad

image_effects:
  cutout: stylized
  stroke: colored-solid | double
  filter: high-saturation

typography:
  decorated: stroke-text | shadow-3d
  tags: bubble | ribbon
  direction: horizontal | curved

decorations:
  emphasis: star-burst | exclamation
  background: solid-saturated | dots
  doodles: stars-sparkles | confetti | squiggles
  frames: none
```

## 色板

| 角色 | 颜色 | Hex |
|------|--------|-----|
| 主色 | 亮红、黄、蓝、绿 | #F56565, #ECC94B, #4299E1, #48BB78 |
| 背景 | 白、浅灰 | #FFFFFF, #F7FAFC |
| 点缀 | 霓虹粉、电光紫 | #FF69B4, #9F7AEA |

## 视觉元素

- 粗形状、对话气泡
- 漫画式效果、星爆
- 动感、有张力的构图
- 高能量装饰

## 字体与排版

- 动感、有活力的手写描边字
- 大胆的配色
- 俏皮、有表现力的字形

## 最佳版式搭配

| 版式 | 适配度 | 适用场景 |
|--------|---------------|----------|
| sparse | ✓✓ | 重磅预告、兴奋感 |
| balanced | ✓✓ | 趣味教程 |
| dense | ✓ | 信息密集 |
| list | ✓✓ | 趣味清单 |
| comparison | ✓✓ | 动感对比 |
| flow | ✓ | 有节奏的流程 |

## 最适合

- 兴奋感预告
- 冷知识、趣味事实
- 互动感强的教程
- 娱乐向内容
- 年轻向内容
