# 画布与版式

小红书信息图的核心画布规格与布局网格。

## 宽高比

| 名称 | 比例 | 像素 | 说明 |
|------|-------|--------|------|
| portrait-3-4 | 3:4 | 1242×1660 | 小红书流量最高（推荐） |
| square | 1:1 | 1242×1242 | 次选推荐 |
| portrait-2-3 | 2:3 | 1242×1863 | 更高竖版 |

**默认**：portrait-3-4，以获得最佳互动。

## 安全区

避免将关键内容放在以下区域：

| 区域 | 位置 | 原因 |
|------|----------|--------|
| bottom-overlay | 底部 10% | 移动端标题栏遮挡 |
| top-right | 右上角 | 点赞/分享按钮遮挡 |
| bottom-right | 右下角 | 水印常见位置 |

```
┌─────────────────────────────┐
│                 [like/share]│  ← 右上角：避开
│                             │
│                             │
│      ✓ 安全内容区            │
│                             │
│                             │
│  [标题栏遮挡区]              │  ← 底部 10%：避开关键信息
└─────────────────────────────┘
```

## 网格布局

### 按信息密度

| 布局 | 信息密度 | 留白 | 要点/图 | 最适合 |
|--------|--------------|------------|--------------|----------|
| sparse | 低 | 60-70% | 1-2 | 封面、金句、强冲击文案 |
| balanced | 中 | 40-50% | 3-4 | 常规内容、教程 |
| dense | 高 | 20-30% | 5-8 | 知识卡、速查表 |

### 按结构类型

| 布局 | 结构 | 条目数 | 最适合 |
|--------|-----------|-------|----------|
| list | 纵向列举 | 4-7 | 排行、清单、步骤指南 |
| comparison | 左 vs 右 | 2 块 | 前后对比、利弊 |
| flow | 连接节点 | 3-6 步 | 流程、时间线、工作流 |
| mindmap | 中心放射 | 4-8 分支 | 概念图、头脑风暴、主题总览 |
| quadrant | 四宫格 | 4 块 | SWOT、优先级矩阵、分类 |

## 按页面位置选布局

| 位置 | 推荐布局 | 原因 |
|----------|-------------------|-----|
| Cover | sparse | 视觉冲击力最大，标题清晰 |
| Setup | balanced | 交代背景但不拥挤 |
| Core | balanced/dense/list | 视内容密度而定 |
| Payoff | balanced/list | 结论清晰 |
| Ending | sparse | CTA 干净，收尾好记 |

## 宫格单元

多元素构图时：

| 名称 | 格数 | 用途 |
|------|-------|----------|
| single | 1 | 主视觉，冲击最强 |
| dual | 2 | 前后对比、对照 |
| triptych | 3 | 步骤、流程 |
| quad | 4 | 产品展示 |
| six-grid | 6 | 清单、合集 |
| nine-grid | 9 | 多图拼贴 |

## 视觉平衡

### Sparse 布局
- 单一视觉焦点居中
- 四周留白
- 对称构图

### Balanced 布局
- 标题偏上
- 下方内容均匀分布
- 层次清晰

### Dense 布局
- 规整网格
- 区块边界清楚
- 紧凑但可读

### List 布局
- 条目左对齐
- 序号/项目符号层级清楚
- 条目格式统一

### Comparison 布局
- 左右对称
- 视觉对比明确
- 中间有分隔

### Flow 布局
- 方向流（上→下或左→右）
- 节点用箭头连接
- 进程指示清楚

### Mindmap 布局
- 中心主题节点
- 向外放射分支
- 层级子分支
- 有机曲线连接

### Quadrant 布局
- 四宫格（2×2）
- 坐标轴标签清楚
- 每象限内容区分明确
- 可选用环形变体表示循环
