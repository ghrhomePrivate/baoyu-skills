# 装饰素材

小红书信息图的视觉点缀与装饰元素。

## Emphasis Marks（强调标记）

用于突出特定内容的元素。

| 名称 | 说明 | 用途 |
|------|-------------|----------|
| red-arrow | 红色箭头指向目标 | 产品卖点、重点 |
| circle-mark | 圆圈高亮标注 | 突出细节 |
| underline | 直线或波浪下划线 | 文字强调 |
| star-burst | 星爆效果 | 特惠、吸睛 |
| checkmark | 对勾符号 | 已完成、优点 |
| cross-mark | 叉号 | 缺点、避免项 |
| exclamation | 感叹号装饰 | 重要提醒 |
| question | 问号装饰 | FAQ、引发好奇 |
| numbering | 带圈数字 | 步骤、排名 |
| bracket | 括号高亮 | 分组、强调 |

| 名称 | 说明 | 用途 |
|------|-------------|----------|
| solid-saturated | 高饱和纯色 | 大胆、有活力 |
| solid-pastel | 柔和粉彩纯色 | 可爱、温和 |
| gradient-linear | 线性渐变 | 现代、动感 |
| gradient-radial | 径向渐变 | 聚光感 |
| frosted-glass | 磨砂玻璃模糊 | 分层叠放 |
| paper-texture | 纸张或手工纸纹理 | 手作感 |
| fabric-texture | 布料纹理 | 温馨、触感 |
| chalkboard | 黑板纹理 | 教育类内容 |
| grid | 淡网格 | 规整、有条理 |
| dots | 波点 | 俏皮、复古 |

## 风格专属装饰

### Cute 风格
- 爱心、星星、闪光
- 丝带、贴纸风元素
- 可爱角色元素

### Notion 风格
- 简单线条涂鸦
- 几何、简笔画小人
- 大量留白、装饰极少

### Warm 风格
- 阳光射线、咖啡杯、温馨小物
- 暖光效果
- 友好、亲切的装饰

### Fresh 风格
- 植物叶、云、水滴
- 简单几何
- 通透、有呼吸感的构图

### Bold 风格
- 感叹号、箭头
- 警示图标、强形状
- 高对比元素

### Pop 风格
- 大胆形状、对话气泡
- 漫画感效果、星爆
- 动感、高能量装饰

### Retro 风格
- 半色调网点、复古徽章
- 经典图标、胶带效果
- 做旧纹理叠加

### Chalkboard 风格
- 粉笔灰效果
- 手绘涂鸦
- 数学公式、简单图标

### Screen-Print 风格
- 粗剪影、几何形
- 半色调网点、印刷颗粒
- 不用涂鸦——靠负空间说话
- stencil 切割边缘、色块分界
- 复古海报式边框处理
