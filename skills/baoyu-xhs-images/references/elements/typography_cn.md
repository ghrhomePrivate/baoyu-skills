# 字体与排版体系

小红书信息图的文本样式元素。

## Decorated Text（花字）

用于强调与视觉吸引力的花字处理。

| 名称 | 说明 | 用途 |
|------|-------------|----------|
| gradient | 渐变填色 | 标题强调、现代感 |
| stroke-text | 描边空心字 | 封面大标题、高可见度 |
| shadow-3d | 3D 阴影/挤出效果 | 关键词、立体感 |
| highlight | 荧光笔涂抹感 | 关键信息、要点 |
| neon | 霓虹发光 | 科技内容、夜景氛围 |
| handwritten | 真实手写风 | 个人感、轻松 |
| bubble | 圆润膨胀字形 | 可爱、俏皮内容 |
| brush | 笔刷纹理 | 艺术感、动感 |

## Tags & Labels（标签）

结构化文字容器。

| 名称 | 说明 | 用途 |
|------|-------------|----------|
| black-white | 黑底白字 | 品牌名、价格、分类 |
| white-black | 白底黑字 | 干净标签、极简 |
| bubble | 对话气泡形 | 对话、批注、要点引出 |
| pointer | 箭头+标签 | 产品标注、指向卖点 |
| ribbon | 丝带/横幅形 | 特惠、亮点 |
| stamp | 印章风格 | 背书、推荐 |
| pill | 胶囊圆角条 | 标签、分类、关键词 |

## 文字层级

视觉层级建议字号。

| 层级 | 作用 | 相对字号 | 样式 |
|-------|------|---------------|-------|
| H1 | 主标题 | 100% | 粗体、花字 |
| H2 | 章节标题 | 70-80% | 半粗 |
| H3 | 小节标题 | 50-60% | 中等字重 |
| Body | 正文 | 40-50% | 常规 |
| Caption | 小字说明 | 30-35% | 较细 |

## 文字方向

| 方向 | 说明 | 用途 |
|-----------|-------------|----------|
| horizontal | 常规从左到右 | 多数内容默认 |
| vertical | 竖排自上而下 | 杂志风、传统中文排版 |
| curved | 沿曲线排字 | 装饰、绕形 |
| diagonal | 倾斜文字 | 动感构图 |

## 文字效果

| 效果 | 说明 | 用途 |
|--------|-------------|----------|
| shadow | 文字投影 | 复杂背景上可读性 |
| outline | 字形外轮廓 | 高对比可见度 |
| glow | 文字柔光 | 梦幻、强调 |
| underline-wavy | 波浪下划线 | 俏皮强调 |
| strikethrough | 删除线 | 前后对比、订正 |

## 语言与排版注意

### 中文
- 标点：「」（）、。！？
- 字间：汉字之间一般不加空格
- 行高：可读性建议 1.5–1.8 倍

### 中英混排
- 中文环境下的英文：注意基线一致
- 数字：统一用 lining 或 old-style，全文一致

## 风格专属字体

### Cute 风格
- 圆润、泡泡感手写
- 柔和阴影、俏皮装饰
- 粉/粉彩点缀文字

### Notion 风格
- 干净的手写感字母
- 简单无衬线标签
- 装饰极少

### Bold 风格
- 带阴影的冲击感手写
- 高对比配色
- 粗描边

### Chalkboard 风格
- 全文字粉笔纹理
- 保留手写不完美感
- 多色粉笔变化
