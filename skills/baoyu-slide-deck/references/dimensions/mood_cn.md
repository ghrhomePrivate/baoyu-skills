# 色调维度

色温和配色风格。

## 选项

| 选项 | 色温 | 配色风格 | 适用场景 |
|--------|-------------------|---------------|----------|
| `professional` | 冷中性 | 藏青、金色、结构灰 | 商业、投资、企业 |
| `warm` | 暖色 | 大地色、橙色、自然色 | 教育、友好、亲和 |
| `cool` | 冷色 | 蓝色、灰色、青色、蓝绿色 | 技术、数据、分析 |
| `vibrant` | 多样 | 高饱和度、大胆色彩 | 营销、创意、吸引注意力 |
| `dark` | 深色 | 深色背景配明亮点缀 | 娱乐、游戏、氛围感 |
| `neutral` | 中性 | 最少色彩、灰度为主 | 高管、极简、精致 |

## 渲染指南

### professional

- 克制使用强调色
- 金色仅用于强调
- 干净、机构化的感觉
- 均衡的对比度

### warm

- 大量使用暖色调
- 自然、亲和的色彩
- 颜色之间柔和的过渡
- 温馨的氛围

### cool

- 以蓝色为主的配色
- 精确的技术用色
- 高对比度保证清晰
- 分析性、可信赖的感觉

### vibrant

- 大胆的色彩组合
- 全程高饱和度
- 动态的色彩对比
- 充满活力的视觉感

### dark

- 深色背景占主导
- 强调色在深色上突出
- 发光/光效效果
- 电影感氛围

### neutral

- 最少的颜色使用
- 字体排印承载重量
- 灰度层次
- 最大化的精致感

## 组合注意事项

| 色调 | 最佳搭配 | 避免搭配 |
|------|-----------------|------------|
| professional | clean 纹理、geometric 字体 | organic 纹理、handwritten |
| warm | organic 纹理、humanist 字体 | pixel 纹理、minimal 密度 |
| cool | grid 纹理、technical 字体 | paper 纹理、handwritten |
| vibrant | pixel/organic 纹理、editorial 字体 | neutral 色调重叠 |
| dark | clean/pixel 纹理、technical 字体 | paper 纹理 |
| neutral | clean 纹理、geometric 字体 | organic 纹理、vibrant 元素 |
