# 字体维度

标题和正文的文字样式。

## 选项

| 选项 | 标题风格 | 正文风格 | 适用场景 |
|--------|----------------|------------|----------|
| `geometric` | 现代无衬线、干净棱角 | 干净无衬线 | 企业、科技、现代 |
| `humanist` | 友好无衬线、温暖曲线 | 易读无衬线 | 教育、普通受众 |
| `handwritten` | 马克笔/毛笔、有机感 | 随意手写或印刷体 | 创意、素描、友好 |
| `editorial` | 粗体衬线/无衬线混合、杂志风 | 经典衬线 | 主题演讲、杂志、高端 |
| `technical` | 等宽装饰、精确 | 干净无衬线 | 开发者、数据、工程 |

## 渲染指南

### geometric

**标题**：现代几何无衬线字体，带有干净棱角和一致的笔画宽度。参考 Futura、Avenir 或 Proxima Nova 的感觉。粗体至半粗体字重。O、G 字符呈完美圆形。

**正文**：为可读性优化的干净无衬线字体。常规字重。一致的 x 高度。充足的字间距。

**特征**：
- 字形的数学精度
- 一致的笔画宽度
- 曲线的完美几何
- 现代、权威的气质

### humanist

**标题**：友好的无衬线字体，带有微妙的笔画变化。参考 Frutiger、Open Sans 或 Myriad 的感觉。中等至半粗体字重。温暖、亲和的字形。

**正文**：易读的人文无衬线字体。舒适的行高。轻微的书法影响。

**特征**：
- 温暖、亲和的感觉
- 微妙的笔画对比
- 开放的字腔保证可读性
- 自然、人性化的触感

### handwritten

**标题**：粗体手写马克笔或毛笔字。粗笔画带有机感边缘。略微不均匀的基线。渲染为实际的手绘字母。

**正文**：清晰的手写风格，模拟笔记。随意但清晰可读。字形自然变化。

**特征**：
- 有机、不完美的字形
- 可见的笔刷/钢笔特征
- 随意、个人化的感觉
- 非计算机字体——实际绘制的字母

### editorial

**标题**：粗体衬线或高对比无衬线。杂志封面风格。戏剧性的大小对比。参考 Playfair Display、Didot 或粗体窄版无衬线的感觉。

**正文**：用于长篇阅读的经典衬线字体。优雅、精致的字形。传统出版品质。

**特征**：
- 高对比度（粗/细笔画）
- 戏剧性的标题
- 精致的气质
- 高端、出版品质

### technical

**标题**：干净无衬线配等宽装饰，用于数据/代码。精确、工程化的外观。参考 SF Mono 用于代码、Inter 用于标题的感觉。

**正文**：为技术内容优化的干净无衬线字体。数字和代码使用固定宽度。

**特征**：
- 数据元素使用等宽字体
- 精确对齐
- 数字清晰区分（0 vs O、1 vs l）
- 工程精度

## 组合注意事项

| 字体 | 最佳搭配 | 避免搭配 |
|------------|-----------------|------------|
| geometric | clean 纹理、professional/neutral 色调 | organic 纹理 |
| humanist | organic/clean 纹理、warm 色调 | pixel 纹理 |
| handwritten | organic/paper 纹理、warm/vibrant 色调 | grid 纹理、professional 色调 |
| editorial | clean 纹理、vibrant/professional 色调 | pixel 纹理 |
| technical | grid/clean 纹理、cool/dark 色调 | paper 纹理、warm 色调 |
