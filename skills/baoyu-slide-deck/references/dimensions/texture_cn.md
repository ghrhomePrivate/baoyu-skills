# 纹理维度

视觉纹理和背景处理。

## 选项

| 选项 | 背景 | 视觉元素 | 适用场景 |
|--------|------------|-----------------|----------|
| `clean` | 纯色实底，无纹理 | 干净线条、几何形状 | 高管、极简、企业 |
| `grid` | 微妙网格叠加 | 网格线、示意图、技术图表 | 技术、架构、工程 |
| `organic` | 柔和纹理、手绘感 | 笔触、水彩、素描线条 | 创意、教育、友好 |
| `pixel` | 像素块、8-bit 美学 | 像素艺术、复古游戏元素 | 游戏、开发者、怀旧 |
| `paper` | 做旧/纹理纸张 | 复古元素、印章、岁月痕迹 | 历史、传统、叙事 |

## 渲染指南

### clean

- 纯色背景，无可见纹理
- 所有元素边缘清晰锐利
- 数字精度和清晰度
- 最大对比度保证可读性

### grid

- 轻微网格叠加（5-10% 不透明度）
- 工程纸或蓝图感
- 对齐参考线可见但微妙
- 技术制图美学

### organic

- 纸张纹理或画布质感
- 不完美的边缘、自然变化
- 手绘色彩填充
- 随意、亲和的感觉

### pixel

- 可见的像素网格（粗块状，非精细）
- 8-bit 色彩方案美学
- 锯齿边缘（无抗锯齿）
- 复古游戏 UI 元素

### paper

- 做旧纸张纹理（微妙褶皱、变色）
- 复古印刷痕迹
- 棕褐色或暖色调
- 历史文件感

## 组合注意事项

| 纹理 | 最佳搭配 | 避免搭配 |
|---------|-----------------|------------|
| clean | professional、neutral 色调 | handwritten 字体 |
| grid | cool、professional 色调 | handwritten、vibrant 色调 |
| organic | warm、vibrant 色调 | technical 字体 |
| pixel | vibrant、dark 色调 | editorial 字体 |
| paper | warm 色调 | geometric 字体、minimal 密度 |
