# 密度维度

每张幻灯片的信息密度。

## 选项

| 选项 | 内容/幻灯片 | 留白 | 适用场景 |
|--------|---------------|------------|----------|
| `minimal` | 单一焦点 | 最大 | 高管简报、主题演讲、情感冲击 |
| `balanced` | 2-3 个要点 | 标准 | 通用演示文稿、混合受众 |
| `dense` | 多个数据点 | 紧凑 | 数据密集、技术评审、详细分析 |

## 渲染指南

### minimal

- 每张幻灯片一个主要观点
- 大型视觉元素占主导
- 最少文字（标题 + 最多 1-2 行）
- 充裕的边距（距边缘 15% 以上）
- 元素之间最大呼吸空间
- 让单一元素承载全部重量

**原则**：
- "一张幻灯片，一条信息"
- 视觉 > 文字
- 空白是有意为之的
- 每个元素都必须有存在价值

### balanced

- 每张幻灯片 2-3 个要点
- 标准边距（距边缘 10%）
- 均衡的文字/视觉比例
- 清晰的层次和支持细节
- 舒适的阅读体验

**原则**：
- 主要观点 + 支持上下文
- 视觉补充文字
- 有结构但不拥挤
- 适合多样化受众

### dense

- 可包含多个数据点
- 紧凑边距（距边缘 5-8%）
- 信息丰富的布局
- 图表、表格、详细注释
- 假设受众高度专注

**原则**：
- 数据比装饰更有说服力
- 注释解释洞察
- 留白是策略性的
- 每个像素都有用途

## 受众 → 密度映射

| 受众 | 推荐密度 |
|----------|-------------------|
| 高管 | minimal |
| 初学者 | minimal 到 balanced |
| 普通受众 | balanced |
| 中级 | balanced |
| 专家 | balanced 到 dense |

## 幻灯片类型 → 密度指南

| 幻灯片类型 | 推荐密度 |
|------------|-------------------|
| 封面/标题 | minimal |
| 章节分隔 | minimal |
| 引言/冲击 | minimal |
| 议程/概览 | balanced |
| 内容/分析 | balanced 或 dense |
| 总结/要点 | balanced |
| 数据/指标 | dense |

## 各密度内容指南

### minimal

| 元素 | 指导方针 |
|---------|-----------|
| 标题 | 大号（等效 40-60pt） |
| 正文 | 最少或无 |
| 要点 | 最多 0-2 个 |
| 视觉元素 | 1 个主导元素 |
| 图表/数据 | 仅 1 个关键统计数据 |

### balanced

| 元素 | 指导方针 |
|---------|-----------|
| 标题 | 中大号（等效 32-48pt） |
| 正文 | 2-4 行 |
| 要点 | 2-4 个 |
| 视觉元素 | 1-2 个元素 |
| 图表/数据 | 简单图表可以 |

### dense

| 元素 | 指导方针 |
|---------|-----------|
| 标题 | 中号（等效 24-36pt） |
| 正文 | 可多段落 |
| 要点 | 4-6+ 个 |
| 视觉元素 | 可多个 |
| 图表/数据 | 复杂图表、表格可以 |

## 组合注意事项

| 密度 | 最佳搭配 | 避免搭配 |
|---------|-----------------|------------|
| minimal | neutral 色调、geometric 字体 | 数据密集的内容 |
| balanced | 任何色调/字体 | 极端情况（太空或太满） |
| dense | cool 色调、technical 字体 | handwritten 字体、organic 纹理 |
